# Services REST API Reference

Manages services of an application.

## Methods


- [`ListServices`](./ListServices.md): Lists all the services in the application.

- [`GetService`](./GetService.md): Gets the current configuration of the specified service.

- [`UpdateService`](./UpdateService.md): Updates the configuration of the specified service.

- [`DeleteService`](./DeleteService.md): Deletes the specified service and all enclosed versions.
